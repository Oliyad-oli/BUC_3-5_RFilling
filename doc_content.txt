Use Case Description
The system allows administrators to configure reminder schedules, templates, channels (email, SMS, portal, print), and escalation rules for tax filing reminders. These settings ensure taxpayers are notified of obligations in a timely and compliant manner.
Actors
System Administrator, Tax Authority Staff
Pre-conditions
User has administrator access rights.Reminder templates and communication channels are available in the system.
Triggers
The tax authority decides to set up or modify filing reminder configurations.
Flow of Events
The system administrator logs into the system.
The system administrator navigates to the Reminder Configuration module.
The system administrator sets frequency and schedule of reminders (e.g., days before/after due date).
The system administrator selects applicable channels (email, SMS, portal, print).
The system administrator designs and configures templates for reminders, letters, and notices.
The system administrator defines severity levels and escalation rules.
The system administrator saves and activates the configuration.
Alternate Flow
AF-01 Invalid Template: If templates are incomplete/invalid, system notifies administrator and blocks activation.AF-02 Channel Unavailable: If a channel (e.g., SMS gateway) is offline, system logs the error and notifies administrator.
Post Conditions
Success: Filing reminder configuration saved and active.Failure: Configuration not applied; administrator notified.
Exceptions
EX-01 System Down: System unavailable → administrator cannot configure reminders.EX-02 Database Error: Configuration not saved → system logs and escalates to IT support.
Information Requirements
Assumptions
Communication infrastructure (SMS, email, portal) is integrated and available.
Reminder templates are predefined and customizable.
Notes and Issues
This use case ensures timely and legally compliant communication with taxpayers regarding filing obligations.
Business Rules
Reminder frequencies must comply with tax legislation.
At least one active channel must be selected for configuration to be valid.
Configurations require administrator approval before activation.

Integrated Tax Administration System, External Ministries and Agencies

Triggers
